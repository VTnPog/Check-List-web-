# ✅ Check List Web

A simple and interactive checklist web app built with HTML, CSS and JavaScript. First time using this languages btw.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Clean and simple interface

## Technologies/Languages Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Font Awesome

## Project Structure

```
Check-List-web-
│
├── index.html
├── css/
│   └── style.css
└── js/
    └── control.js
```

## How to Use

Clone the repository:

```
git clone https://github.com/VTnPog/Check-List-web-.git
```

Then open:

```
index.html
```
